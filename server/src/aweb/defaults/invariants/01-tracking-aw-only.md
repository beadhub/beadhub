id: tracking.aw-only
title: Use aw for coordination tracking
---

Track coordination state in aweb with `aw`. Do not create markdown TODO lists or side trackers that drift from the shared system.

- Run `aw roles show` for the active invariants and role guidance
- Use `aw work ready` to find unblocked work
- Use `aw work active` to see work already in progress
- Link discovered follow-up work to the parent task in the shared system
- Store planning docs in `history/` if needed
